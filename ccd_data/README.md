
This will help you understand how CCD works and whether you need to clean the CCD glass.

Connect the sensor to the PC and run the python file ccd_data.py(which in the github) in the cmd shell .
also make sure you have installed the software python.
 
for example:

```
C:\Users\admin>python D:\Github\bdwidth\ccd_clean\ccd_data.py com6 500000
```
